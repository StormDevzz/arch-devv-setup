# Утилита настройки разработки для Arch Linux

Эта консольная утилита автоматически установит часто используемые инструменты разработки и запустит нужные службы.

## Как собрать и запустить
1. Установите .NET SDK (если ещё не установлен):
   ```bash
   sudo pacman -S --noconfirm dotnet-sdk
   ```
2. Сборка проекта:
   ```bash
   dotnet build -c Release
   ```
3. Запуск утилиты (может потребовать ввод пароля sudo):
   ```bash
   ./bin/Release/net7.0/devsetup
   ```

## Что делает утилита
- Устанавливает пакетный набор через `pacman`.
- Включает и стартует Docker, PostgreSQL и Redis.
- Запускает Python‑скрипт для установки pip‑пакетов.

## Структура проекта
```
devsetup/
├─ Program.cs      # Основной код
├─ Config.cs       # Списки пакетов и сервисов
├─ Utils.cs        # Вспомогательная функция RunCommand
├─ Logger.cs       # Простейший логгер
├─ setup.py        # Python‑скрипт для pip‑пакетов
├─ devsetup.csproj # .NET проект
└─ README.md       # Этот файл
```
